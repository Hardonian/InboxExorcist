# Customer Support Macros

## Gmail did not connect

Gmail did not finish connecting. Please retry from "Exorcise my inbox." If the error mentions scopes, Google did not grant the access needed to create reversible labels and filters.

## Did InboxExorcist delete my emails?

No. InboxExorcist does not delete by default. Quieting archives matching messages and applies `InboxExorcist/Quieted`, so they remain searchable in Gmail.

## Unsubscribe did not work

InboxExorcist only reports confirmed unsubscribe when a safe standards-based path completes. If unsubscribe was unavailable or unsafe, we apply a reversible Gmail filter instead.

## Undo my changes

Use Settings -> Action history -> Undo latest. Undo removes filters created by InboxExorcist. Gmail messages already archived can still be found under the applied label and All Mail.

## Delete my data

Use Settings -> Delete my data. This removes your InboxExorcist connection, scans, candidates, actions, filters, unsubscribe attempts, allowlists/blocklists, and audit records.
