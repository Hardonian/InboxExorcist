## Creator Brief

Make it feel like relief, not enterprise software. Show the preview before action, the safety skips, and undo. Never claim all unsubscribe attempts succeed. Never show real personal inbox data.

## Before/After Screen Recording Guide

Use a seeded demo Gmail account. Start on Promotions with fake brands. Run scan. Pause on preview. Quiet selected senders. Show `InboxExorcist/Quieted`. Show action log and undo.

## Privacy-Safe Demo Data

Use invented domains like `deals.example`, `newsletter.example`, and `orders.example`. Blur account email. Do not show message bodies, snippets, unsubscribe links, order numbers, bank names, or personal senders.
